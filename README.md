# dbmsproject-hms

This project is aimed to automate the hospital database system. This project is developed mainly to administrate the details of doctors, patients and doctor's appointments with the patients in Jins Hospital. The purpose of the project entitled as HOSPITAL Database SYSTEM is to computerize the Front Office Management of Hospital to develop software which is user friendly, simple, fast, and cost - effective. It provides various features, which complement the information system and increase the productivity of the system.It deals with the collection of patient's information, feedback, appointments, etc and the collection of the doctorâ€™s information along with schedules. Traditionally, it was done manually.The main function of the system is to register and store patient details and doctor details and retrieve these details as and when required, and also to manipulate these details meaningfully. Furthermore, access to the software is password protected, enabling only the admin or the users added by the admin to login.
